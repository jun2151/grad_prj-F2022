3. util.py 설명 
- 다음과 같은 기능을 파이썬 코드로 작성하고 데트스하였습니다 

- 1. sample_rate 변환 
. wav file의 sample_rate를 변환하는 기능입니다
. 사용 package
  import librosa, soundfile 
  from scipy.io import wavfile
. python 패키지 설치 
  pip install librosa
  pip install wavfile

- 2. 음성데이터 자르기 
. 음성파일의 일부분만을 파일로 생성하는 기능입니다 
. 사용 package
  import librosa, soundfile

- 3. stereo to mono channel 변환 
. 음성파일 channel을 stereo에서 mono 파일로 변환하는 기능입니다
. 사용 package
  from pydub import AudioSegment
. 해당기능 실행을 위하여 ffmpeg(fast forward mpeg) sw 설치가 필요합니다 
  ffmpeg는 모든 동영상, 음악, 사진 포맷들의 디코딩과 인코딩을 목표로 만들어지고 있는 오픈 소스 프로젝트입니다
  ffmpeg를 다운로드하여 PC에 설치 
  실행경로 설정: set PATH=%PATH%;<경로>\ffmpeg\bin
. pydub 패키지 추가 설치  
  pip install pydub

- 4. 마이크로 입력받은 음성을 파일로 저장
. 마이크로 입력받은 음성데이터를 파일로 저장하는 기능입니다
. 사용 package
  import pyaudio
  import wave

- 5. wav file을 읽어서 play sound 실행 
. wav file을 읽어서 음성을 스피커에 나오게 하는 기능입니다
. 사용 package
  import pyaudio
  import wave

- 6. wav file을 읽어서 chunk 단위로 음성 데이터 확인
. 사용 package
  from scipy.io import wavfile

- 7. wav file을 읽어서 play sound 실행함 
. wav file을 읽어서 음성을 스피커에 나오게 하는 기능입니다
. 사용 package
  from scipy.io import wavfile
  import pyaudio
