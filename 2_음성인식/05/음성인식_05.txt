
1. 일반환경에서 음성데이터를 인식하여 text로 변환하는 기능이 잘 작동되나
   졸업작품에서 사용할 지하철 안내방송은 전동차 소음등으로 
   음성인식이 되지 않는것을 알게되었습니다
   지하철안내방송 음성데이터의 소음을 줄여서 음성인식을 할수있게 하는
   소음제거 오픈소스를 조사하고 테스트 하였습니다

. 참조한 url:  https://github.com/dodiku/noise_reduction
. sox 설치 필요함 
  sox is a cross-platform command line utility 
  that can convert various formats of computer audio files in to other formats
. 추가설치 패키지  
  pip install pysndfx
  pip install python_speech_features
. 결과: 소음 제거 효과 없음

4. 소음제거 코드 적용 결과
소음제거 소스를 조사하여 참조코드를 적용하여 테스트 해보았습니다
